## 📊 Widgets Disponibles

### 🏆 BarComm Pro
Dashboard commercial avec suivi des métriques réseaux sociaux, gestion des devis et contrats événementiels.

🔗 **[Accéder à BarComm Pro](https://cocktailprivategit.github.io/barcomm-pro/)**

Fonctionnalités :
- Suivi Instagram, TikTok, LinkedIn
- Tableau de bord devis & contrats
- Métriques de performance commerciale
- Export/Import des données JSON

---

### 💰 BarMan Finance Pro
Gestion financière complète avec rapports analytiques et visualisations graphiques.

🔗 **[Accéder à BarMan Finance Pro](https://cocktailprivategit.github.io/barman-finance/)**

Fonctionnalités :
- Suivi des revenus et dépenses
- Graphiques analytiques interactifs
- Rapports mensuels et annuels
- Export/Import des données JSON

## 🚀 Démarrage Rapide

### Option 1 — Utilisation directe (recommandé)
Cliquez simplement sur les liens ci-dessus. Aucune installation requise.

### Option 2 — Cloner en local
```bash
# 1. Cloner le dépôt
git clone https://github.com/cocktailprivategit/widgets.git
cd widgets

# 2. Ouvrir un widget
open src/barcomm-pro/index.html
# ou
open src/barman-finance/index.html
```

Pour la configuration avancée, consultez [SETUP.md](./docs/SETUP.md).
